
# Как обнулить всю статистику?
Запустить файл delete_all_stats.py


# Где указываются данные от базы данных?
Все данные от бд указываются в файле config.py, в переменных: DB_HOST, DB_PASSWORD, DB_USER, DB_NAME, DB_PORT


# Где указывается токен бота для вк версии?
В файле config.py, в переменной VK_TOKEN


# Где указывается токен бота для телеграмм версии?
В файле config.py, в переменной TG_TOKEN


# Как дать человеку права добавлять задания?
Нужно открыть файл config.py, найти в нем массив REDACTORS и вписать в него Telegram ID человека


# Как сделать человека админом в вк версии?
Нужно открыть файл config.py, найти в нем массив VK_ADMINS и вписать в него VK ID человека


# Как сделать человека админом в телеграмм версии?
Нужно открыть файл config.py, найти в нем массив TG_ADMINS и вписать в него Telegram ID человека
